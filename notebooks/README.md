# Notebooks

This folder contains Jupyter notebooks created during the Data Analytics Internship.

## Notebooks Included

- iris_task.ipynb  
  Classification analysis using the Iris dataset with EDA and ML models.

- stock_task.ipynb  
  Time series analysis and visualization of stock market data.

- sentiment_task.ipynb  
  NLP-based sentiment analysis on text data.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- NLP libraries
